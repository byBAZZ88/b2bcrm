1\. НАЧАЛО РАБОТЫ

Запуск

Дважды кликните по файлу b2bcrm\_v1.2.exe. Программа откроется в окне. Никаких паролей для входа не требуется — база данных уже привязана к вашему магазину руководителем.

Главное окно

В верхней части окна отображается название вашего магазина. Ниже расположены четыре вкладки.

📊 Дашборд — оперативные показатели. Используется каждое утро для планирования дня.

🖥️ Рабочий стол — поиск и работа с клиентами. Основной инструмент в течение всего дня.

📥 Загрузка данных — импорт продаж из ViSet. Используется каждый понедельник.

⚙️ Настройки — доступ только у руководителя.

2\. РАБОЧИЙ СТОЛ — ПОИСК КЛИЕНТОВ

Это ваша главная вкладка. Здесь вы ищете клиентов и работаете с ними.

Как искать

В поле «Поиск по любым полям базы данных...» можно ввести любую информацию: часть названия организации (например, «Строй»), ИНН (например, «7806»), телефон (например, «921»), фамилию представителя (например, «Иванов»), слово из комментария или описания.

Минимальная длина запроса — 3 символа.

При необходимости выберите статус из выпадающего списка (например, «Новый») для фильтрации.

Нажмите кнопку «🔍 Искать» или клавишу Enter.

Результаты поиска

В левой части экрана появится список найденных клиентов. Для каждого показаны название организации, ИНН и текущий статус. Статусы выделены цветом: красный — Новый, оранжевый — Переговоры, синий — Выставлен счет, зелёный — Постоянный, серый — все остальные.

Просмотр карточки клиента

Кликните левой кнопкой мыши по строке в списке. В правой части откроется полная карточка клиента со всеми данными: название, статус, ИНН, КПП, код ViSet, магазин-владелец, ФИО представителя, телефон(ы), email, тип деятельности, канал привлечения, сумма продаж, дата последней продажи, дата создания, дата следующего контакта (если установлена), описание и последний комментарий.

Все текстовые поля можно выделять мышью и копировать через Ctrl+C.

3\. ДОБАВЛЕНИЕ НОВОГО КЛИЕНТА

Когда добавлять

Вам позвонил новый клиент, которого нет в базе. Или вы нашли нового клиента через выставку, рекомендацию, интернет. Сразу после контакта создайте карточку, чтобы не потерять информацию.

Как добавить

Нажмите кнопку «+ Новый контрагент». Откроется форма с несколькими разделами.

Основные реквизиты.

Название — полное наименование организации или ИП (обязательно). Например: ООО «Ромашка» или ИП Иванов И.И.

ИНН — 10 цифр для юридических лиц, 12 цифр для ИП (обязательно). Программа сама определит тип по длине ИНН и предложит переключатель «Это ИП».

КПП — 9 цифр для юридических лиц (обязательно для ЮЛ). Для ИП поле блокируется автоматически.

Контактные данные.

Телефон — 11 цифр, начиная с 7. Можно указать несколько номеров через запятую с пробелом, например: «79261234567, 79261234568». Каждый номер проверяется на корректность.

Email — адрес электронной почты в стандартном формате.

ФИО представителя — с кем разговаривали, кто принимает решения.

Дополнительно.

Канал привлечения — выберите из списка или введите свой (например, «Холодный звонок», «Выставка», «Рекомендация партнёра»).

ID ViSet — можно оставить пустым, заполнится автоматически после первой покупки.

Описание — до 200 знаков, любая постоянная информация о клиенте. Например: «Крупная строительная компания, работают по 44-ФЗ, требуются закрывающие документы».

Комментарий — суть первого разговора. Например: «Перезвонить в четверг, уточнить объёмы».

Нажмите «✅ Сохранить контрагента». Программа проверит обязательные поля и сохранит карточку.

4\. ФИКСАЦИЯ КОНТАКТА — ГЛАВНЫЙ ПРОЦЕСС

Каждый разговор с клиентом должен быть зафиксирован в программе. Это позволяет не терять историю и видеть, на каком этапе находится работа.

Порядок действий

Найдите клиента через поиск.

Кликните по нему в списке — откроется карточка.

Нажмите кнопку «🔧 Открыть карточку для работы». Вместо режима просмотра откроется режим редактирования с двумя блоками.

Блок «Редактирование реквизитов». Здесь можно обновить контактные данные: телефон (один или несколько через запятую), email, ФИО представителя, тип деятельности (выбрать из списка или ввести свой), описание (любая информация о клиенте, не привязанная к конкретному звонку).

Блок «Фиксация нового контакта». Выберите новый статус клиента из выпадающего списка. При необходимости установите дату следующего контакта через встроенный календарь (актуально для статусов «Сезонный», «Смена локации», «Отказ от сотрудничества», «Контакт по дате»). В поле «Комментарий» кратко опишите результат разговора.

Нажмите «✅ Зафиксировать контакт».

Требования при переводе в «Переговоры»

При переводе клиента из статуса «Новый» в статус «Переговоры» обязательно должны быть заполнены четыре поля: телефон, email, ФИО представителя и тип деятельности. Если хотя бы одно из этих полей не заполнено, программа покажет список того, что нужно дозаполнить, и не даст сохранить изменения.

Требования для статуса «Контакт по дате»

Для этого статуса обязательно указать дату следующего контакта. Без даты сохранение будет заблокировано.

Что писать в комментарии

Комментарий — это краткая суть разговора. Он меняется при каждом контакте.

Хорошие примеры: «Уточнили объёмы поставки, ждём спецификацию до пятницы», «Выслали КП на почту, обещал посмотреть к среде», «Оплатил счёт №245, отгрузка завтра утром», «Отказался, не устраивает цена на позицию Х».

Плохие примеры: «Позвонил», «Всё хорошо», «Ок». Такие комментарии не несут информации для будущей работы.

Что писать в описании

Описание — это постоянная информация о клиенте, не связанная с конкретным звонком. Оно сохраняется между контактами.

Хорошие примеры: «Крупная строительная компания, филиал в Москве», «Работают по 44-ФЗ, требуются закрывающие документы», «Предпочитают доставку по вторникам и четвергам», «Контактное лицо меняется раз в полгода».

5\. ДАШБОРД — КОНТРОЛЬ БАЗЫ

Вкладка «📊 Дашборд» показывает, сколько клиентов требуют вашего внимания прямо сейчас. Она состоит из цветных плиток-счётчиков.

Плитки дашборда

🔴 Новые. Клиенты, с которыми ещё ни разу не связывались. Требуют немедленного контакта. Отрабатывать в первую очередь.

🟡 Переговоры. Клиенты, с которыми нет связи более 7 дней. Они могут остыть и уйти к конкурентам. Позвонить, предложить акцию, продвинуть к сделке.

⚡ Выставлен счет. Клиенты, которым отправили счёт, но оплата не поступила более 3 дней. Срочно напомнить об оплате и уточнить дату.

🟢 Постоянные. Клиенты, с которыми нет связи более 14 дней. Плановый обзвон: напомнить о себе, узнать, какая нужна помощь, сообщить о новинках.

🌐 Смена локации. Клиенты, которые переехали в другой регион. Проверить раз в месяц — не вернулся ли клиент работать в наш регион.

❄️ Сезонный. Клиенты, работающие только в определённый сезон. Контакт раз в 90 дней: поздравить с праздниками, узнать планы на следующий сезон.

📅 Контакт по дате. Клиенты с гибким графиком закупок, звонок строго по указанной дате.

⚠️ Реанимация. Клиенты, которые не покупали более 90 дней. Попытаться выяснить причину и вернуть.

📋 Всего. Общее количество клиентов в базе.

Как работать с дашбордом

Откройте вкладку «📊 Дашборд». Посмотрите на плитки — чем ярче цвет, тем срочнее требуется внимание.

Кликните по плитке — вы автоматически перейдёте на Рабочий стол с готовым списком клиентов для отработки. По очереди открывайте карточки, звоните, фиксируйте контакт. Возвращайтесь к дашборду и переходите к следующей плитке.

Рекомендуемый порядок отработки. Сначала красная плитка «Новые» — те, кому вы ещё не звонили. Затем «Выставлен счет» — срочно напомнить об оплате. Далее «Контакт по дате» — если дата звонка сегодня. Потом «Переговоры» — не дать остыть. «Постоянные» — поддержать отношения. «Сезонный» — в межсезонье. «Смена локации» — раз в месяц. «Реанимация» — когда есть свободное время.

6\. ЕЖЕНЕДЕЛЬНЫЙ ИМПОРТ ПРОДАЖ

Каждый понедельник необходимо загрузить данные о продажах за прошедшую неделю из учётной системы ViSet.

Порядок действий

Выгрузите из ViSet отчёт по продажам за прошедшую неделю в формате Excel.

Перейдите на вкладку «📥 Загрузка данных». В верхней части экрана будет указана отчётная неделя — с понедельника по воскресенье. Убедитесь, что период правильный.

Нажмите «📂 Выбрать файл» и выберите Excel-файл из ViSet. Нажмите «🚀 Запустить импорт». Дождитесь завершения — прогресс-бар покажет ход выполнения.

После завершения программа покажет результат: сколько строк обновлено (существующие клиенты — обновились суммы и даты продаж), сколько создано новых (клиенты, которых ещё нет в базе — они автоматически создадутся со статусом «Новый»), сколько пропущено (строки без названия или с нулевой суммой).

7\. ТИПИЧНЫЙ РАБОЧИЙ ДЕНЬ

Утро понедельника

Откройте программу. Перейдите на вкладку «📥 Загрузка данных» и загрузите отчёт ViSet за прошлую неделю. Затем перейдите на «📊 Дашборд» и отработайте плитку «Новые» — все новые клиенты требуют первого контакта.

Ежедневно

Откройте «📊 Дашборд». Отработайте клиентов по порядку приоритета: Новые → Выставлен счет → Переговоры → Постоянные. В течение дня фиксируйте каждый звонок через кнопку «✅ Зафиксировать контакт». При поступлении новых клиентов сразу создавайте карточки.

8\. СТАТУСЫ КЛИЕНТОВ — ПОЛНЫЙ СПРАВОЧНИК

🔴 Новый. Карточка только что создана. С клиентом ещё не работали. Действие: позвонить в течение дня. Следующий статус: Переговоры.

🟡 Переговоры. Клиент заинтересован, идёт обсуждение условий. Действие: выставить счёт, прислать коммерческое предложение. Следующий статус: Выставлен счет.

⚡ Выставлен счет. Счёт отправлен, ожидается оплата. Действие: контролировать оплату, напоминать. Следующий статус: Постоянный.

🟢 Постоянный. Клиент регулярно покупает. Действие: поддерживать контакт, информировать об акциях и новинках.

🌐 Смена локации. Клиент переехал в другой регион и прекратил закупки. Действие: проверять раз в месяц — не вернулся ли. Следующий статус: Постоянный (если вернулся).

💀 Ликвидация. Организация закрылась, ИНН исключён из реестра. Действие: исключён из работы.

🚫 Отказ от сотрудничества. Клиент прямо отказался работать. Причины могут быть разные. Действие: не тратить время, исключён из активной работы.

❄️ Сезонный. Клиент покупает только в определённый сезон (например, строительные работы только летом). Действие: поздравить с праздниками, узнать планы на следующий сезон. Следующий статус: Постоянный (в начале сезона).

📅 Контакт по дате. Клиент с гибким графиком закупок — может покупать каждые 3 дня или раз в 2 месяца. Действие: позвонить строго в указанную дату. Если не дозвонились — перенести дату на другую. Без указания новой даты статус сменить нельзя. Следующий статус: любой.

9\. ВАЖНЫЕ ЗАМЕЧАНИЯ

Фиксируйте каждый контакт. Если не записали — считайте, что не звонили. Через неделю вы уже не вспомните детали разговора.

Не удаляйте файл crm\_data.db из папки с программой. Это база данных со всеми клиентами, историей контактов и суммами продаж.

Не передавайте программу и базу третьим лицам. База содержит персональные данные клиентов (телефоны, email, ИНН).

Пароль от настроек хранится только у руководителя. Если вы не можете войти в настройки — обратитесь к руководителю.

При любых ошибках в программе сообщите руководителю. Не пытайтесь исправить что-то самостоятельно.

10\. ВКЛАДКА «ОТЧЁТЫ»

Вкладка «📈 Отчёты» предназначена для контроля качества работы и анализа клиентской базы. Она разделена на три зоны.

Верхняя панель — выбор периода

В верхней части находится фильтр периода. Выберите один из вариантов:

День — отчёт за конкретную дату. Выберите дату из календаря.

Неделя — отчёт за выбранную неделю. Укажите номер недели (от 1 до 53) и год.

Месяц — отчёт за конкретный месяц. Выберите месяц из списка и год.

Квартал — отчёт за квартал (1, 2, 3 или 4) и год.

Год — отчёт за весь год.

Произвольный — отчёт за любой период. Укажите даты «с» и «по» через календари.

После выбора периода нажмите кнопку «🔄 Обновить». Показатели справа пересчитаются.

Левая панель — выгрузка

Кнопка «📥 Excel» формирует полноценный отчёт в формате Excel и сохраняет его на Рабочий стол. Файл содержит несколько листов с подробной аналитикой.

Кнопка «📄 PDF» станет доступна в следующих версиях.

Под кнопками отображается время последнего обновления данных.

Правая панель — оперативные показатели

После нажатия «Обновить» отображаются шесть ключевых показателей за выбранный период:

Контактов за период — общее количество зафиксированных контактов со всеми клиентами.

Новых клиентов — количество клиентов, впервые внесённых в базу за период.

Переведено в Переговоры — количество клиентов, с которыми начались переговоры.

Выставлено счетов — количество клиентов, которым выставлены счета.

Стали Постоянными — количество клиентов, перешедших в статус постоянных.

Отказов и ушедших — сумма клиентов, отказавшихся от сотрудничества, сменивших локацию или ликвидированных.

Ниже крупных показателей отображаются два дополнительных:

Среднее время в статусе «Новый» — сколько дней в среднем проходит от создания карточки до первого перехода в «Переговоры». Показывает скорость реакции менеджера.

Просрочено контактов (SLA) — количество клиентов, по которым превышен допустимый срок без контакта. Требует немедленного внимания.

Последние контакты

В нижней части панели отображается таблица с пятью последними зафиксированными контактами. Для каждого показаны: дата и время, название клиента, изменение статуса (с какого на какой), текст комментария.

11\. КАК ЧИТАТЬ ВЫГРУЖЕННЫЙ ОТЧЁТ

После нажатия кнопки «📥 Excel» на Рабочем столе создаётся файл с именем «Отчёт\_дата\_начала\_дата\_конца.xlsx». Откройте его. Файл содержит семь листов.

Лист «Сводка»

Дублирует оперативные показатели из правой панели программы. Все цифры за выбранный период в одной таблице.

Лист «Воронка»

Показывает конверсию клиентов, созданных в выбранном периоде, по этапам воронки продаж.

Колонка «Этап» — название этапа (Новый, Переговоры, Выставлен счет, Постоянный).

Колонка «Количество» — сколько клиентов из созданных в периоде достигли данного этапа.

Колонка «Конверсия от когорты» — процент от общего числа новых клиентов, дошедших до этапа.

Пример чтения: создано 100 новых клиентов, из них 40 дошли до Переговоров (40%), 15 до Выставлен счет (15%), 8 стали Постоянными (8%). Если конверсия низкая — проверьте качество обработки на ранних этапах.

Лист «По статусам»

Распределение всех клиентов в базе по статусам на текущий момент. Две колонки: Статус и Количество. Позволяет оценить общую картину базы.

Лист «Каналы привлечения»

Показывает, из каких источников пришли новые клиенты за выбранный период. Отсортировано по убыванию. Колонки:

Канал — название источника (Холодный звонок, Выставка, Сайт компании и т.д.).

Новых клиентов — сколько клиентов пришло через этот канал.

Анализируйте этот лист для оценки эффективности рекламы и источников.

Лист «Качество данных»

Проблемы в базе, требующие исправления:

Без телефона — клиенты без контактного телефона. Невозможно позвонить.

Без Email — клиенты без электронной почты. Невозможно отправить КП.

Без типа деятельности — клиенты без указания сферы деятельности. Важно для сегментации.

Без контактов — клиенты в статусе «Новый», с которыми ни разу не связывались. Требуют первоочередного внимания.

Лист «Динамика базы»

Изменение клиентской базы по месяцам за последний год. Колонки:

Месяц — год и месяц.

Новые — сколько клиентов создано в этом месяце.

Ушедшие — сколько клиентов переведены в статусы «Отказ от сотрудничества», «Смена локации» или «Ликвидация».

Прирост — разница между новыми и ушедшими. Положительное число — база растёт. Отрицательное — сокращается.

Здоровая динамика: прирост положительный, ушедших меньше новых. Если ушедших больше — проверьте причины оттока.
